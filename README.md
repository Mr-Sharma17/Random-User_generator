# About
The Random User Generator is a web application where user can get the details of a random user.

### Features :
>
> - rendering real-time data form server
> - clean UI
> - animation
>

### Tech-Stacks used :
>
> - HTML
> - CSS
> - Vanilla JS
>

### Concepts used :
>
> - API calling
> - CSS animations
> - DOM Manipulation
> - listening events using JS
>
